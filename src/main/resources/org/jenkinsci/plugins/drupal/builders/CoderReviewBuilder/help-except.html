Specify modules/themes that should not be reviewed, relative to the Drupal root directory.

For instance if the Drupal root is 'drupal' and if you want to review only custom code then you might want to exclude contributed projects: 

    drupal/sites/all/modules/contrib/**,
    drupal/sites/all/themes/contrib/**,
    drupal/modules/**,
    drupal/themes/**,
    drupal/profiles/**

This field supports FileSet includes.
